# Video Game Sales Analysis

## Project Overview

The Video Game Sales Analysis project focuses on exploring global gaming industry trends by combining sales data, engagement metrics, and ratings to uncover meaningful business insights.
The goal of this project is to demonstrate a complete end to end analytics workflow from raw data preparation to SQL analysis and interactive Power BI dashboards.

This project analyzes

Genre performance across global markets
Platform dominance and regional differences
Sales trends over time
Impact of user ratings and engagement
Market growth patterns within the gaming industry

The workflow reflects a real world analytics pipeline designed for business decision making.

## Project Workflow

### Data Cleaning and Exploration using Python

Standardized column names and data types
Handled missing values and duplicates
Performed exploratory analysis on engagement and sales data
Prepared datasets for relational modeling

### Database Design and SQL Analysis

Created structured relational tables
Merged engagement and sales datasets
Wrote analytical queries to answer business questions such as
Top genres by global sales
Regional performance comparison
Sales trends over time
Platform rating analysis

### Data Visualization using Power BI

Built professional dashboards including
Executive overview of global performance
Regional and platform based analysis
Sales trend visualization
KPI tracking for decision making

## Key Insights

Action and Sports genres generate the highest global sales
North America contributes the largest share of revenue followed by Europe and Japan
Console platforms such as PS2 X360 and PS3 dominate historical performance
The industry experienced rapid growth between 2000 and 2010 followed by a decline in later years
Engagement metrics like plays and wishlists help explain market demand patterns
